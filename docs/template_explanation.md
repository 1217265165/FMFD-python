# 仿真模板说明 (Template T1-T9 Explanation)

## 模板概述

仿真系统使用9种故障模板来生成不同类型的故障特征。每个模板模拟特定的物理故障机制。

## 模板详细说明

### T1: smooth_shift (平滑偏移)
- **故障类型**: 幅度失准 / 参考电平失准
- **物理机制**: 整体增益或参考电平漂移
- **表现形式**: 整条幅频曲线上下平移
- **参数范围**:
  - 轻度: ±0.04 ~ ±0.10 dB
  - 中度: ±0.10 ~ ±0.20 dB
  - 重度: ±0.20 ~ ±0.34 dB
- **关联模块**: 衰减器、校准源、存储器

### T2: tilt_rolloff (倾斜滚降)
- **故障类型**: 幅度失准
- **物理机制**: 低频到高频的增益斜率变化
- **表现形式**: 曲线呈现斜坡特征
- **参数范围**:
  - 轻度: -0.03 ~ -0.08 dB/归一化频率
  - 中度: -0.06 ~ -0.14 dB/归一化频率
  - 重度: -0.10 ~ -0.20 dB/归一化频率
- **关联模块**: 低频段前置低通滤波器、低频段第一混频器、中频放大器

### T3: stable_ripple (稳定纹波)
- **故障类型**: 幅度失准 / 参考电平失准
- **物理机制**: 周期性幅度波动，常由阻抗匹配不良引起
- **表现形式**: 正弦波形叠加在基线上
- **参数范围**:
  - 幅度: 0.03 ~ 0.18 dB
  - 周期: 0.04 ~ 0.16 归一化频率
- **关联模块**: 低频段前置低通滤波器、存储器、数字RBW

### T4: step_fixed (固定台阶)
- **故障类型**: 参考电平失准
- **物理机制**: 切换点处的校准不连续
- **表现形式**: 曲线某处突然跳变
- **参数范围**:
  - 台阶高度: ±0.05 ~ ±0.30 dB
  - 台阶位置: 频率范围的25%~75%处
- **关联模块**: 存储器、校准信号开关

### T5: spike_sparse (稀疏尖峰)
- **故障类型**: 幅度失准
- **物理机制**: 随机干扰或元器件不稳定
- **表现形式**: 随机位置的突变点
- **参数范围**:
  - 尖峰数量: 4~22个
  - 尖峰幅度: ±0.08 ~ ±0.30 dB
- **关联模块**: 衰减器、本振混频组件、电源模块

### T6: scatter_thick (散射加厚)
- **故障类型**: 幅度失准
- **物理机制**: 噪声增大，通常与电源纹波相关
- **表现形式**: 曲线整体变"毛躁"
- **参数范围**:
  - 噪声标准差: 0.02 ~ 0.12 dB
- **关联模块**: 电源模块、数字检波器、VBW滤波器

### T7: quant_grain (量化颗粒)
- **故障类型**: 参考电平失准
- **物理机制**: ADC量化误差或数字处理精度不足
- **表现形式**: 曲线呈现阶梯状颗粒感
- **参数范围**:
  - 量化步长: 0.008 ~ 0.030 dB
- **关联模块**: ADC、数字放大器

### T8: peak_track (峰值追踪)
- **故障类型**: 频率失准
- **物理机制**: 本振频率不准或锁相环问题
- **表现形式**: 峰值位置偏移、展宽或凹陷
- **子类型**:
  - spike: 尖峰型 (35%概率)
  - dense: 密集型 (45%概率)
  - hole: 凹陷型 (20%概率)
- **关联模块**: 时钟振荡器、时钟合成与同步网络、本振源（谐波发生器）

### T9: band_offset (频段偏移)
- **故障类型**: 多种
- **物理机制**: 特定频段的选择性故障
- **表现形式**: 某一频段整体偏移
- **参数范围**:
  - 偏移幅度: ±0.06 ~ ±0.24 dB
  - 频段宽度: 0.2 ~ 0.9 归一化频率
- **关联模块**: 多种，取决于具体频段

## 模板与故障类型的映射关系

| 故障类型 | 主要模板 | 次要模板 |
|----------|----------|----------|
| normal | 无 | 无 |
| amp_error | T1, T2, T5, T6 | T3 |
| freq_error | T8 | T9 |
| ref_error | T1, T3, T4, T7 | T6 |
